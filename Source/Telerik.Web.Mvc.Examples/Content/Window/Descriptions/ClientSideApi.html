This example shows how to use the client-side API of Telerik Window for ASP.NET MVC.

To use the client-side API you should first retrieve the window client-side object: 

<script type="text/javascript">
function getWindow(){
    //"Window" is the value specified by the Name() method.
    var window = $("#Window").data("tWindow");
    return window;
}
</script>

Available client-side methods:

  • You can open window with the open() method.
  • You can close window with the close() method.
  • You can re-load window content with the refresh() method.
  • You can maximize window with the maximize() method.
  • You can restore window size with the restore() method.
