This example shows the client-side events supported by Telerik Window for ASP.NET MVC.

The window has the following events:

  • OnLoad - raised when the component is initialized
  • OnOpen (cancellable) - raised when the window is opened
  • OnClose (cancellable) - raised when the window is closed
  • OnRefresh - raised when the window content is re-loaded

You can subscribe to each of the events in the following ways:

  • With a pre-defined javascript function 
    <%= Html.Telerik().Window()
        .Name("Window")
        .ClientEvents(events => events
            .OnLoad("OnLoad")
        )
%>

<script type="text/javascript">
    function OnLoad(e) {
        // your code
    }
</script>
  • With an inline function, serialized from the server 
    <% Html.Telerik().Window()
       .Name("Window")
       .ClientEvents(events => events
       .OnLoad(() => {%> function(e) { // your code } <%})
       .Render();
%>
  • Purely client-side event registration 
    <%= Html.Telerik().Window().Name("Window") %>

<script type="text/javascript">
    $(document).ready(function() {
        $('#Window').bind('open', function(e) { // your code });
    });
</script>
