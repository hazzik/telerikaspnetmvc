This example shows the accessibility features of Telerik Slider for ASP.NET MVC and Telerik RangeSlider for ASP.NET MVC.

In environments with disabled JavaScript, the slider will fall-back to plain <input> elements with type range.

Telerik Slider for ASP.NET MVC and Telerik RangeSlider for ASP.NET MVC are level A accessible (in compliance with the W3C Web Accessibility Guidelines 2.0).
