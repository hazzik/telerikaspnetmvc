This example shows the filtering feature of Telerik ComboBox for ASP.NET MVC and Telerik AutoComplete for ASP.NET MVC with server binding.

To enable filtering use the Filterable method: 

<%= Html.Telerik().ComboBox()
        .Name("ComboBox")
        .Filterable()
%>

You can also choose between "StartsWith" and "Contains" filter options. 

<%= Html.Telerik().ComboBox()
        .Name("ComboBox")
        .Filterable(filtering =>
        {
            filtering.FilterMode(AutoCompleteFilterMode.StartsWith);
        })
%>

When filtering is disabled (default state), combobox will scroll to the first matched item and highlight it.

Filtering feature of the autocomplete component cannot be disabled.
